# PrestaShop Modules Documentation

This directory contains the modules for PrestaShop, which extend the functionality of the platform. Below are some key points regarding the modules:

## Installation

To install a module, follow these steps:

1. Download the module files.
2. Place the module folder in the `prestashop/modules` directory.
3. Log in to the PrestaShop back office.
4. Navigate to the "Modules" section.
5. Find the module in the list and click "Install".

## Usage

Once installed, you can configure the module settings from the PrestaShop back office. Each module may have its own specific configuration options.
